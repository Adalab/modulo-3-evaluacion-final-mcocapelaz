# Harry Potter Character Finder

Enlace en Git Pages: 

Este proyecto pertenece a la evaluación final del módulo sobre React del bootcamp de programación web de Adalab. 
Se trata de una aplicación web que permite buscar y filtrar personajes del universo de Harry Potter por nombre, casa y género.

## Características

- Búsqueda por nombre de personaje (mínimo 3 caracteres)
- Filtrado por casa: Gryffindor, Slytherin, Hufflepuff, Ravenclaw o todas
- Filtrado por género: femenino, masculino o todos
- Reset de filtros para volver al estado inicial
- Carga dinámica de personajes desde una API pública
- Mensajes de carga y aviso cuando no hay resultados

## Tecnologías 

- React (hooks: useState, useEffect)
- React Router para navegación entre listado y detalle de personajes
- Fetch API para consumir datos externos
- Sass para estilos (archivo App.scss)

## Uso

1. Escribe el nombre (al menos 3 letras) para filtrar personajes por nombre.
2. Selecciona una casa de Hogwarts para filtrar personajes.
3. Selecciona el género para filtrar personajes masculinos, femeninos o todos.
4. Usa el botón Reset para limpiar todos los filtros y volver al estado inicial).
